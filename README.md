# SEO Performance Analysis

## Objective
Analyze SEO and marketing performance data to identify traffic trends,
top-performing pages, and optimization opportunities.

## Tools Used
- Excel / Google Sheets
- SQL (optional)
- Tableau / Looker Studio

## Dataset
Simulated SEO performance data including sessions, impressions, clicks,
CTR, conversions, and keyword rankings over 6 months.

## Key Questions
- Which pages drive the most organic traffic?
- Which keywords have high impressions but low CTR?
- How does traffic change over time?
- Where are conversion drop-offs?

## Analysis Summary
- Organic traffic increased by X% over 6 months
- Blog content accounted for the majority of organic sessions
- Several keywords show strong impressions but weak CTR
